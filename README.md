# recetitas.app

Landing de recetitas.app, una comunidad para descubrir, guardar y compartir recetas de gente real.

## Stack

- Vue 3 + Vite
- Tailwind CSS v4
- Phosphor Icons
- Sites / Cloudflare Worker-compatible build

## Desarrollo local

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

La versión publicada está disponible en [recetitas-app.stherling53.chatgpt.site](https://recetitas-app.stherling53.chatgpt.site).
